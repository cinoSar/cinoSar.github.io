About This Course

This course is designed for PhD students, early-career researchers, and advanced undergraduates who have some prior exposure to statistics but need to rebuild and strengthen their foundations before tackling econometric analysis. If you took a statistics module years ago and feel uncertain about concepts like confidence intervals, p-values, or regression assumptions, this course will help you reconstruct that knowledge with depth and rigour.

How This Course Works

Each module combines three elements:

  1. Conceptual explanation: We build intuition first, connecting ideas to real econometric problems before formalising them.
  2. Python code: Every concept is accompanied by working code (pandas, matplotlib, seaborn, scipy, statsmodels) so you can see the ideas in action.
  3. Exercises: Each module ends with exercises that test both your reasoning and your ability to implement the concepts.

You can read each module as a standalone web page. If you want to run and modify the code yourself, every module has a downloadable Jupyter Notebook and a link to open it directly in Google Colab — no installation required.

Course Structure

The course is organised in four progressive blocks:

  1. Foundations: Descriptive statistics, probability, and distributions — the language and intuition you need before doing any inference.
  2. Inferential Statistics: Sampling, the Central Limit Theorem, confidence intervals, hypothesis testing — the core of statistical reasoning.
  3. Regression & Modelling: Simple and multiple regression, diagnostics, transformations, logistic regression — the bridge to econometrics.
  4. Towards Econometrics: Endogeneity, instrumental variables, panel data, causal inference methods — where statistics meets economic research.
