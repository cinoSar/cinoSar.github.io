About This Course

This course teaches QGIS from the ground up by building one cumulative project with real Padova data — geocoded firm records, zoning polygons, and territorial boundaries. Each lesson adds a new part to the same working file, so you see how project setup, data loading, styling, spatial operations, and export fit together within a coherent workflow.

No prior knowledge of GIS is required. The course can be completed entirely through the QGIS graphical interface; an optional Python sidebar in each lesson offers a scripting route for learners who already know basic programming.

What You Will Build

By the end of the course, you will be able to:

  1. Create and organise a QGIS project properly, with a clear folder structure, reusable styles, and sensible project settings.
  2. Load and inspect tabular and spatial data, including geocoded firm records and polygon layers such as zoning and administrative perimeters.
  3. Style layers clearly and produce readable maps that communicate a spatial question rather than merely display data.
  4. Perform basic and intermediate spatial analysis, using filters, joins, location-based selections, overlays, and other core tools.
  5. Export polished outputs for reports, presentations, or websites.
  6. Automate selected workflows through the QGIS Python Console.

How This Course Works

Each lesson combines four elements:

  1. Guided replication: you reproduce the workflow step by step inside QGIS, building confidence with the software and learning where the main tools are.
  2. Checkpoint: at the end of the guided section, you verify that your project matches the expected result before moving on.
  3. Open-ended task: each lesson ends with a related but slightly different challenge. Early tasks closely reinforce the guided work; later tasks ask you to solve a similar problem using a stronger or more flexible QGIS tool.
  4. Python sidebar: every lesson includes a scripting component that progressively builds your ability to automate GIS tasks. These sections are optional but cumulative — if you follow them, you will have a working Python-based workflow by the end of the course.

Course Structure

The course is organised in four progressive blocks:

  1. Foundations: installing QGIS, understanding the interface, setting up projects correctly, and learning how GIS data is structured.
  2. Working with Layers and Attributes: loading spatial and tabular data, checking coordinate systems, styling maps, filtering features, and joining information across layers.
  3. Spatial Analysis: using location-based tools such as selection by location, clipping, buffering, and overlays to answer real analytical questions.
  4. Outputs and Automation: creating polished map layouts, exporting results, and automating repeated tasks through the QGIS Python Console.

Two paths through the course

The GUI path is self-contained: you can complete every lesson without writing a single line of code. The Python sidebar runs in parallel, building scripting skills lesson by lesson. You can follow either path independently, or both together.
